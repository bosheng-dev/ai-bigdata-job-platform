# AI大数据就业资讯平台

🎯 专为高职人工智能与大数据专业学生打造的就业信息聚合平台

## 核心特性

- ✅ **真实可靠**: 所有信息来自经过验证的招聘平台
- ✅ **时效保障**: 仅展示30天内发布的职位，过期自动下架
- ✅ **精准分类**: 按AI工程师、大数据工程师、数据分析师等类别智能归类
- ✅ **实时更新**: 支持定时自动抓取最新职位

## 技术栈

- **后端**: Node.js + Express
- **数据库**: SQLite
- **爬虫**: Axios + Cheerio
- **前端**: 原生 HTML/CSS/JS

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 手动触发爬虫
npm run crawl

# 运行测试
npm test
```

## API 接口

| 接口 | 说明 |
|------|------|
| `GET /api/jobs` | 获取职位列表（支持分页、筛选、搜索） |
| `GET /api/jobs/:id` | 获取职位详情 |
| `GET /api/jobs/stats/categories` | 获取分类统计 |
| `POST /api/admin/crawl` | 手动触发爬虫（管理员） |

## 信息来源

- 主流招聘平台: BOSS直聘、智联招聘、前程无忧
- 企业官方招聘页面
- 高校就业信息网
- 政府人才服务平台

## 功能特性

### 用户端
- 📱 **现代化界面**: 响应式设计，支持移动端
- 🔍 **智能筛选**: 按分类、城市、工作类型、关键词筛选
- 📧 **职位订阅**: 支持邮箱订阅，每日推送匹配职位
- 📊 **实时统计**: 展示平台职位、企业、城市数据

### 管理后台
- 📝 **职位管理**: 添加、编辑、删除、审核职位
- 📈 **数据统计**: 分类统计、来源分析
- 🔔 **订阅管理**: 查看订阅用户、发送记录

### 数据服务
- 🤖 **自动抓取**: 每日凌晨自动抓取新职位
- 🧹 **自动清理**: 过期职位自动下架
- 📧 **自动推荐**: 每日上午推送订阅邮件
